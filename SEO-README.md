# SEO Setup — Digital View

## ✅ Hecho

- Meta tags completos (title, description, keywords, robots, geo)
- Open Graph (Facebook, WhatsApp, LinkedIn)
- Twitter Cards
- Canonical URL → `https://digitalviewstrategies.com/`
- JSON-LD: Organization + LocalBusiness + WebSite + Service + FAQ
- `robots.txt`
- `sitemap.xml`
- `site.webmanifest` (PWA básico)
- Lazy loading en fotos del equipo
- Alt text descriptivo en imágenes
- WhatsApp link en contacto + footer
- Dirección, horarios y país en footer
- Año dinámico en footer
- ARIA labels en redes sociales

## 🔧 Pendiente (necesita archivos del usuario)

Colocá estos archivos en la raíz del proyecto (`web-digital/`):

| Archivo | Tamaño | De dónde sacarlo |
|---|---|---|
| `logo.png` | mín 512×512 | Pasámelo |
| `favicon.svg` | vector | Convertir logo a SVG |
| `favicon-32x32.png` | 32×32 | Generar en realfavicongenerator.net |
| `favicon-16x16.png` | 16×16 | idem |
| `apple-touch-icon.png` | 180×180 | idem |
| `android-chrome-192x192.png` | 192×192 | idem |
| `android-chrome-512x512.png` | 512×512 | idem |
| `og-image.jpg` | 1200×630 | Diseño con logo + slogan |

**Generador rápido de favicons:** subí `logo.png` a https://realfavicongenerator.net y descargá el paquete — ya viene con todos los formatos.

## 🚀 Próximos pasos (cuando me pases datos del paso 6)

1. **Google Analytics 4** — pegame el ID `G-XXXXXXX`
2. **Meta Pixel** — pegame el Pixel ID
3. **Google Search Console** — verificar el dominio (te dejo guía cuando llegue)
4. **Google Business Profile** — clave para Local SEO en San Isidro

## 📊 Acciones post-deploy

1. Subir sitio a `digitalviewstrategies.com`
2. **Google Search Console** → enviar `sitemap.xml`
3. **Bing Webmaster Tools** → enviar `sitemap.xml`
4. Validar rich snippets: https://search.google.com/test/rich-results
5. Test mobile: https://search.google.com/test/mobile-friendly
6. PageSpeed: https://pagespeed.web.dev/
7. Crear ficha **Google Business Profile** en San Isidro
